Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

/*
После выполнения программы мы получаем deadlock, потому что range по каналам итерируется до тех пор, пока он не будет закрыт. В данном случае канал не был закрыт, поэтому, когда не осталось той горутины, которая в канал пишет, main зависает в deadlock.
Чтобы это исправить, достаточно закрыть канал в безымянной горутине, которая в канал пишет, когда писать будет уже не надо, достаточно добавить defer close(ch) в начале этой горутины и дедлока не произойдет, а программа завершится как надо.
*/
