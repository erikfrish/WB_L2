Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

/*
В обоих случаях отложенная функция выполняется после завершения выполнения функции, но до возврата из нее.
В функции test() именованное возвращаемое значение x изменяется  безымянной функцией, вызванной через defer, до возврата из функции test, поэтому результат равен измененному значению x — 2.
В функции anotherTest, хотя отложенная функция также изменяет x, но в return уже передано другое значение, это изменение не влияет на возвращаемое значение функции, так как она возвращает локальное значение x — 1.

В первом случае x объявлен в сигнатуре функции, а возвращяемое значение x определяется после выполнения блока defer, а во втором случае return x выполняется до выполнения блока defer, и хотя x после изменяется, на возвращаемое значение это не повлияет.
*/
```
